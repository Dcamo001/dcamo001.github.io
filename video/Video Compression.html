Welcome to PS2PDF

Welcome to ps2pdf.com

We are the web's premier online converter of Portable Document Format (PDF) files. Our site allows you to make a PDF compatible file without loading any special software. All it takes to generate a PDF compatible file is to upload a postscript compatible file to our web site and then click on the resulting link. See the FAQ page for more details.

Our aim is to make it simple for light and moderate users to generate PDF files. For heavy use we recommend the freely available AFPL Ghostscript . This is the same software we use behind our web pages. We also recommend GSview as the best way to view and edit PDF files. Files converted on this site are also viewable with Adobe Acrobat® software.
